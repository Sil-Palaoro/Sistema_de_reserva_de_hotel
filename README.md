README - Sistema de Reserva de Hotel
Descripción
Este proyecto consiste en el desarrollo de un sistema básico de reserva de hotel realizado en JavaScript para ejecutar por consola.
El sistema permite administrar clientes, habitaciones y reservas mediante un menú interactivo.
Objetivo del Proyecto
Desarrollar un sistema simple que permita simular el funcionamiento básico de un hotel, aplicando conceptos de programación, estructuras de datos y lógica.
Funcionalidades del Sistema
Gestión de Clientes
Registrar clientes
Almacenar nombre y documento
Gestión de Habitaciones
Visualizar habitaciones disponibles
Tipos de habitaciones:
Simple
Doble
Suite
Gestión de Reservas
Crear reservas
Modificar reservas
Cancelar reservas
Ver reservas registradas
Operaciones del Hotel
Check-in
Check-out
Gestión de Estados
Las reservas pueden tener los siguientes estados:
Confirmada
Cancelada
En curso
Finalizada
Tecnologías Utilizadas
JavaScript
Consola del navegador / Node.js
